# BidProperty

Astro + Tailwind CSS v4 + React 19 property bidding website.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Pages

- `/` home
- `/properties` property listing
- `/properties/[id]` property details
- `/my-bids` bid dashboard mockup
- `/account` account prompt
- `/sign-in` sign in
- `/sign-up` sign up
